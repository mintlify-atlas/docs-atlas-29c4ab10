---
title: "Create Project"
description: "Create a new construction project with an optional 3D model or image-based reference."
---

## Overview

Creates a new project and persists it to the database. When `reference_type` is `3d_model` and a `model_url` is provided, the model data is saved to local disk and the stored URL is updated to a server-relative path under `/uploads/`.

`POST /api/projects`

## Request body

<ParamField body="name" type="string" required>
  Display name for the project. Used to derive the filename when saving a 3D model to disk.
</ParamField>

<ParamField body="reference_type" type="string" required>
  Spatial reference method for zones in this project. Must be one of:
  - `images` — zones are linked to reference photographs.
  - `3d_model` — zones are mapped onto an uploaded 3D model.
</ParamField>

<ParamField body="model_url" type="string">
  Base64-encoded GLB file (data URL) or a URL to the 3D model. Only used when `reference_type` is `3d_model`. The server saves the file locally and stores the resulting path instead of the raw data URL.
</ParamField>

<ParamField body="target_completion_date" type="string">
  Scheduled completion date in ISO 8601 format (e.g., `2025-12-31`). Stored as `null` if not provided.
</ParamField>

<ParamField body="budget" type="number">
  Total project budget. Defaults to `0` if omitted.
</ParamField>

## Response

Returns the newly created project object with HTTP status `201`.

<ResponseField name="id" type="string" required>
  Unique identifier assigned to the new project (MongoDB ObjectId as a string).
</ResponseField>

<ResponseField name="name" type="string" required>
  Project name as provided in the request.
</ResponseField>

<ResponseField name="reference_type" type="string" required>
  Reference type as provided in the request (`images` or `3d_model`).
</ResponseField>

<ResponseField name="model_url" type="string | null">
  Server-relative path to the stored 3D model (e.g., `/uploads/My_Project_1714000000000.glb`), or `null` when not applicable.
</ResponseField>

<ResponseField name="target_completion_date" type="string | null">
  ISO 8601 date string or `null`.
</ResponseField>

<ResponseField name="overall_progress" type="number" required>
  Always `0` on creation.
</ResponseField>

<ResponseField name="budget" type="number" required>
  Budget value stored for this project.
</ResponseField>

<ResponseField name="created_at" type="string" required>
  ISO 8601 timestamp set automatically at creation.
</ResponseField>

## Examples

<CodeGroup>

```bash cURL
curl -X POST https://your-domain.com/api/projects \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Tower Block A",
    "reference_type": "3d_model",
    "model_url": "data:model/gltf-binary;base64,<BASE64_DATA>",
    "target_completion_date": "2025-12-31",
    "budget": 1500000
  }'
```

```typescript TypeScript
const response = await fetch('https://your-domain.com/api/projects', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    name: 'Tower Block A',
    reference_type: '3d_model',
    model_url: 'data:model/gltf-binary;base64,<BASE64_DATA>',
    target_completion_date: '2025-12-31',
    budget: 1500000,
  }),
});

const project = await response.json();
console.log(project.id); // "6627a1b2c3d4e5f6a7b8c9d0"
```

</CodeGroup>

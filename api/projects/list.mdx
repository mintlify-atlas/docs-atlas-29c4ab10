---
title: "List Projects"
description: "Retrieve all construction projects sorted by creation date."
---

## Overview

Returns an array of all projects in the system, ordered from most recently created to oldest.

`GET /api/projects`

## Request

No query parameters or request body required.

## Response

Returns a JSON array of project objects.

<ResponseField name="id" type="string" required>
  Unique identifier for the project (MongoDB ObjectId serialized as a string).
</ResponseField>

<ResponseField name="name" type="string" required>
  Display name of the project.
</ResponseField>

<ResponseField name="reference_type" type="string" required>
  How zones within this project are spatially referenced. One of `images` or `3d_model`.
</ResponseField>

<ResponseField name="model_url" type="string | null">
  Path to the uploaded 3D model file (e.g., `/uploads/site_model_1714000000000.glb`). `null` when `reference_type` is `images`.
</ResponseField>

<ResponseField name="target_completion_date" type="string | null">
  ISO 8601 date string for the scheduled completion of the project. `null` if not set.
</ResponseField>

<ResponseField name="overall_progress" type="number" required>
  Aggregate construction progress across all zones, expressed as a percentage from `0` to `100`.
</ResponseField>

<ResponseField name="budget" type="number" required>
  Total project budget. Defaults to `0` if not provided at creation.
</ResponseField>

<ResponseField name="progress_history" type="object[]">
  Chronological list of progress snapshots recorded for this project.

  <Expandable title="properties">
    <ResponseField name="date" type="string" required>
      ISO 8601 timestamp of when this snapshot was recorded.
    </ResponseField>

    <ResponseField name="progress" type="number" required>
      Overall progress value at the time of this snapshot (0–100).
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="created_at" type="string" required>
  ISO 8601 timestamp for when the project was created.
</ResponseField>

## Examples

<CodeGroup>

```bash cURL
curl https://your-domain.com/api/projects
```

```typescript TypeScript
const response = await fetch('https://your-domain.com/api/projects');
const projects = await response.json();

console.log(projects);
// [
//   { id: '6627a1b2c3d4e5f6a7b8c9d0', name: 'Tower Block A', ... },
//   ...
// ]
```

</CodeGroup>

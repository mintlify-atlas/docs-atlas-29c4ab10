---
title: "Delete Project"
description: "Permanently delete a project and all of its associated rooms."
---

## Overview

Deletes a project and cascades the deletion to all rooms belonging to it. This operation cannot be undone.

`DELETE /api/projects/{id}`

<Warning>
  Deleting a project is irreversible. All rooms and their progress entries associated with this project will be permanently removed from the database. There is no soft-delete or recovery mechanism.
</Warning>

## Path parameters

<ParamField path="id" type="string" required>
  The project's unique identifier (MongoDB ObjectId string).
</ParamField>

## Response

Returns a JSON object confirming the deletion.

<ResponseField name="success" type="boolean" required>
  `true` when the project and its rooms were successfully deleted.
</ResponseField>

## Examples

<CodeGroup>

```bash cURL
curl -X DELETE https://your-domain.com/api/projects/6627a1b2c3d4e5f6a7b8c9d0
```

```typescript TypeScript
const id = '6627a1b2c3d4e5f6a7b8c9d0';

const response = await fetch(`https://your-domain.com/api/projects/${id}`, {
  method: 'DELETE',
});

if (!response.ok) {
  const error = await response.json();
  throw new Error(error.error);
}

const result = await response.json();
console.log(result.success); // true
```

</CodeGroup>

## Response example

```json 200
{
  "success": true
}
```

## Error responses

```json 404
{
  "error": "Project not found"
}
```

```json 500
{
  "error": "Failed to connect to database"
}
```
